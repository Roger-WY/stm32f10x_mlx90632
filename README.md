# MLX90632_STM32F10X测试工程

## 介绍

基于 STM21F103 的标准库写的 MLX90632的测试工程，支持 IAR For ARM 9.3及以上版本以及 keil5 MDK

## 软件架构

使用标准库编写，每隔 1s 采集一次温度，并通过单片机的串口1打印

## 使用方法

1. 首先将传感器和STM32F103的开发板通过杜邦线连接

| MLX90632接线 |      |
| :----------- | :--- |
| MLX90632-SCL | PC4  |
| MLX90632-SDA | PC5  |
| 串口接线     |      |
| RXD          | PA9  |
| TXD          | PA10 |
|              |      |

2. 打开 IAR/MDK 的工程，下载程序到开发板中，观察现象即可
